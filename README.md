### RUN
- using python3.10
```bash
pip3.10 install -r requirements.txt
python3.10 main.py
```
### run with nohup

```bash
nohup python3.10 main.py &
```

### install docker and docker compose
```bash
sudo apt-get update
sudo apt-get install docker.io
sudo apt-get install docker-compose
```

### run with docker
```bash
docker-compose up -d --build
```
